# Merienda Haus - Phygital Experience

This is a "Physical-Digital" web experience for Merienda Haus, Berlin.

## Features
- **Contextual UI:** Changes based on time of day (Night Mode after 7 PM) and simulated weather.
- **Bento Grid Layout:** Interactive, responsive grid layout.
- **Wolt Island:** A "Dynamic Island" style floating element for delivery.
- **Instagram Stories:** A social media integration style slider.
- **Social Proof:** Simulated notification-style reviews.

## How to Run
Simply open `src/index.html` in any modern web browser.

## Technologies
- HTML5 (Semantic, SEO Schema)
- CSS3 (Variables, Grid, Flexbox, Animations)
- Vanilla JavaScript (No dependencies)

## Context Simulation
The weather is simulated (randomized on reload) to demonstrate the "Rain/Sun" copy changes.
Night mode is automatic based on your system time (assumed Berlin time zone).
